> Chapter 1: 3D Models and Modelling

Issue Possible Cause Solution
Model won't load in slicer Corrupted or unsupported file Try re-downloading or converting it to .STL
Model is invisible or tiny Incorrect scale or units Rescale the model in slicer (e.g., 10× or 1000×)
Slicer crashes or freezes Complex/high-poly model Reduce file size using Meshmixer or Blender
Gaps or missing sections in sliced preview Model has mesh errors Use Netfabb or the slicer's repair tool
Floating parts or islands Model isn't manifold or has disconnected parts Combine objects or add supports in slicer
